# Blueprint landing page

Marketing site for **Blueprint** — a link-in-bio product.

## Run locally

```bash
cd landing
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

All copy and section data live in `src/content/site.ts`. Update that file when syncing requirements from Linear.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
